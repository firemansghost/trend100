# Project Brain

This directory contains the **project brain** — the canonical documentation that tracks the state, decisions, tasks, and handoffs for the Trend100 Momentum Dashboard project.

## Files

- **PROJECT.md** — Project overview, scope, philosophy, and operating preferences
- **STATUS.md** — Current project state, blockers, and next actions
- **DECISIONS.md** — Record of architectural and design decisions
- **TASK_LOG.md** — Chronological log of completed tasks
- **HANDOFF.md** — Handoff notes for the next session
- **CHECKS.md** — Quality checks and validation criteria

## Usage

When starting a new session, read all files in this directory to understand the current state of the project before making changes.
